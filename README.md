# XO Game (C# WinForms)

A classic XO game developed using C# and Windows Forms.

## Features
- **Dynamic Logic:** Checks for 8 winning combinations.
- **Visual Feedback:** Highlights the winning buttons in green.
- **Smart UI:** Displays current turn and winner status using managed labels.
- **Reset Functionality:** Restarts the game state and UI seamlessly.

## How to Run
1. Clone the repository.
2. Open the `.sln` file in Visual Studio.
3. Press `F5` to build and run.
